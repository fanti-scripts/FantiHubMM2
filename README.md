# 🔪 FANTI HUB — Murder Mystery 2

**Fanti Hub** — это мощный и многофункциональный скрипт для игры Murder Mystery 2. Он создан, чтобы дать тебе полный контроль над игрой с помощью удобного графического меню и широкого набора инструментов.

---

## 🚀 Загрузка и запуск

Просто скопируй и выполни эту команду в своём инжекторе:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/fanti-scripts/FantiHubMM2/main/zapenic_obf.lua"))()
📋 Полный список функций
⚔️ Combat (Боевые)
AimBot — автоматическое наведение прицела на убийцу. Больше никаких промахов!

FOV — регулировка радиуса действия прицела (от 50 до 500).

Show FOV — отображение зоны прицела на экране.

👁️ Visual (Визуальные)
ESP — подсвечивает всех игроков через стены с цветовой индикацией ролей:

🟢 Зелёный — Невиновный

🔵 Синий — Шериф

🔴 Красный — Убийца

GunESP — выделяет выпавшее оружие рамкой и текстом на карте.

Shader Menu — отдельное меню для изменения визуального стиля игры с пресетами Evening и Night.

FPS Boost — повышает производительность, отключая тени, туман и лишние эффекты.

🛠️ Other (Прочее)
Teleport Tool — телепортация в любую точку по клику мыши.

Reset Roles — принудительный сброс цветов в ESP для всех игроков.

Statistics — отображение в реальном времени FPS, пинга, количества игроков и скорости.

Fling Menu — полноценное меню для мульти-флинга с выбором целей (START / STOP).

AntiFling — защита от флинга (отключает коллизии у других игроков).

Walk Speed — быстрая установка скорости ходьбы (16 / 16.5 / 16.7).

🏃 Movement (Движение)
BHOP (Half-Life 2) — система бани-хопа, позволяющая сохранять и наращивать скорость в воздухе.

Spin Presets — моментальное включение вращения персонажа со скоростью 50 или 100.

🎮 Управление
Клавиша	Действие
Right Shift	Открыть / закрыть главное меню
Клик по кнопке FantiHub	Открыть / закрыть главное меню
