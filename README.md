# 🔪 FANTI HUB — Murder Mystery 2

**Fanti Hub** — это многофункциональный скрипт для игры Murder Mystery 2 с удобным графическим меню и большим набором функций. Скрипт защищён обфускацией и распространяется под лицензией GPLv3.

---

## 🚀 Загрузка и запуск

Скопируй и выполни эту команду в своём инжекторе:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/fanti-scripts/FantiHubMM2/main/zapenic_obf.lua"))()
📋 Функционал
⚔️ Combat (Боевые)
AimBot — автоматическое наведение прицела на убийцу.

FOV — регулировка радиуса прицела (от 50 до 500).

Show FOV — отображение зоны прицела на экране.

👁️ Visual (Визуальные)
ESP — показывает игроков через стены с цветовой индикацией:

🟢 Зелёный — Невиновный

🔵 Синий — Шериф

🔴 Красный — Убийца

GunESP — подсвечивает выпавшее оружие рамкой и текстом.

Shader Menu — отдельное меню с шейдерами (Evening / Night).

FPS Boost — повышает производительность, отключая тени, туман и эффекты.

🛠️ Other (Прочее)
Teleport Tool — телепортация по клику мыши.

Reset Roles — сброс цветов в ESP.

Statistics — отображение FPS, пинга, количества игроков и скорости.

Fling Menu — мульти-флинг с выбором целей (START / STOP).

AntiFling — защита от флинга (отключает коллизии у других игроков).

Walk Speed — быстрая установка скорости (16 / 16.5 / 16.7).

🎮 Управление
Клавиша	Действие
Right Shift	Открыть / закрыть меню
Клик по кнопке FantiHub	Открыть / закрыть меню
